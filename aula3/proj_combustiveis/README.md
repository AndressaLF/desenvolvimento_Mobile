# Overview

Nessa aula construimos um aplicativo que calcula o melho combustível que deve ser utilizado entre Alcool e Gasolina.

O aplicativo recebe o valor de entrada dos dois combustíveis, refere ao seu preço, realizar o calculo dividindo o valor do preço do alcool/gasolina.

Se o resultado dessa divisão for maior que 0.70. Ele indica a Gasolina como melhor combustível, caso contrário, Alcool.
